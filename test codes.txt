üìù Notes

    Status LED (LED 0): Reserved for connection status, but can be controlled with "leds" command
    "all" command: Always skips LED 0
    Brightness (b): Range 0-255, defaults to 255 if omitted
    LED index (i): Range 0-9 for 10 LEDs
    Colors: Use hex format #RRGGBB
    Speed: Options are "slow", "medium", "fast" (defaults to "medium")
    Effects auto-stop: When you set colors with "all" or "leds", any running effect stops
